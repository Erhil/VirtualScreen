# Guide 04 - Scripts And Live Tools

Open [[Scripts/random_tavern_event.dms]] and run it from the editor or Scripts tool. Open [[Scripts/screen_audio_demo.dms]] to see a small script that sends content to the player screen and plays audio.

Use Capture from the workspace controls to save a quick note into today's session log. Use Prep Check to scan this world for broken links and missing literal script paths.

Open the HP tool and add a temporary combatant. Save a Table State Snapshot from Actions, change the screen/map/audio state, then load the snapshot.

Use the path picker buttons beside setup fields instead of typing exact paths. Configure a fast slot, keyboard binding, or MIDI binding for a common action if you want a table-control shortcut.
