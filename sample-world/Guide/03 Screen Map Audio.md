# Guide 03 - Screen Map Audio

Open `/screen` in another browser tab or device.

In the Screen tool, use the Display tab to show this guide or [[Media/sample-map.svg]] fullscreen. Stage [[Cards/Clockwork Gull.cs]] as a popup, then show and hide it.

In the Screen tool's Map tab, load [[Media/sample-map.svg]], present it, enable fog, reveal a rectangle, hide part of it again, then try the polygon reveal and polygon hide tools. Add a pin and stop presentation when you are done.

In the Audio tool, play the tiny sample tracks from `.music`: ambient tavern, bard music, and a broken-glass effect. Collapse the Audio tool while playback continues.

Use **Saved Playlists** inside the Audio tool to make a session set without moving files on disk. Saved playlists are DM-local app state stored under `.virtualscreen`, so they do not appear as normal world pages.
