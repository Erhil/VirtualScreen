# Guide 02 - Cards And Tables

Cards are `.cs` JSON files rendered as compact sheets.

Open [[Cards/Basic Character Sheet.cs]] to see typed fields and a sheet layout. Open [[Cards/Computed Character Sheet.cs]] to see safe computed fields such as ability bonuses. Open [[Cards/Item Reference Table.cs]] to see a table section.

Try the `New Card` button in the workspace controls and create a card from a built-in or world-local template. World-local templates live in `.virtualscreen/card-templates` and stay hidden from normal browsing.

CSV tables remain normal files too. Open [[Tables/random-events.csv]] and search for `silver crate`.
